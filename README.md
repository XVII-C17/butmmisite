# Mon Site

Un template moderne pour démarrer votre site web rapidement.

## Fichiers inclus

- **index.html** - Page HTML principale avec structure complète
- **styles.css** - Feuille de style réactive et moderne
- **script.js** - Scripts JavaScript pour les interactions

## Caractéristiques

✅ Design responsive (mobile, tablette, desktop)  
✅ Navigation fluide avec scroll smooth  
✅ Section hero attrayante  
✅ Grille de services  
✅ Formulaire de contact  
✅ Footer  
✅ Animations et transitions  

## Démarrage rapide

1. Ouvrez le fichier `index.html` dans votre navigateur
2. Personnalisez le contenu selon vos besoins
3. Modifiez les couleurs dans `styles.css`
4. Ajoutez vos propres fonctionnalités dans `script.js`

## Personnalisation

### Couleurs principales
- Bleu primaire: `#667eea`
- Violet: `#764ba2`
- Gris foncé: `#2c3e50`

Modifiez ces valeurs dans `styles.css` pour changer le thème.

### Contenu
Remplacez le texte placeholder par votre contenu dans `index.html`.

## Support des navigateurs

- Chrome / Edge (dernière version)
- Firefox (dernière version)
- Safari (dernière version)
